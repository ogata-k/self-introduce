## 数学
実は私は最初から数学が好きだったわけではありません。順番に列挙すると、次のような経歴で数学科に入るほど数学が好きになりました。
1. 小学校までは学問が全体的に得意。
2. 中学では理系科目以外があまり得意なくなる。
3. 高校では、三角関数の応用や充電と積分の関係などを知り、ふと立ち寄った本屋で独習C言語という本に出会いプログラミングと出会う。そこで道具としての数学に興味を持つ。このころ数学ガールを知る。この時点で理系科目以外苦手、その理系科目の中では数学が楽しいと思える程になり、それ以外は学問として好きといった感じに。
4. 行列や線形代数、群などの「代数学」、確率分布や検定などの「確率・統計学」、位相空間や多様体などの「位相幾何学」、一般的に使われる公理としての集合論と肩を並べるほど有用な「圏論」（どれも好き。）などに触れることで抽象数学（数学の中でも抽象化を目指す分野。抽象代数学、多様体、圏論もその一つ。）に興味を持つ。クイバー（有向グラフの一種。）と表現をゼミで学ぶことで表現論にも少し興味を持つ。
5. 2018年7月末に自分が興味を持った数学の分野（代数的構造、クイバー、ホモロジー代数、圏論、多様体）が（代数的）位相幾何学という分野を中心に考えることができることに気づく。さらに統計学や表現論、微分（位相）幾何学にも応用が利くらしい。そして今後数学はこの分野を中心に勉強していくことにすることに決める。
6. 代数的位相幾何学を学ぶうちにほとんどの理論が圏論ないしグラフ的に表現することでわかりやすくなることに気づき、大学四年の10月にしてグラフによる表現論およびグラフの操作論、構造解析などに興味を持ち始める。もちろんホモロジー代数は今後も学んでいくがそれと並行にクイバーの表現論やアーベル圏での一般化されたホモロジー代数をひとまずの目標に今後学んでいくつもり。

## コーヒー
コーヒーを好きになったのは、大学2年か3年の頃にネスカフェの香味焙煎シリーズを買ったのがきっかけです。今までのインスタントコーヒーとは比べ物にならないほどおいしかったのでそこからコーヒーを飲むことにはまりました。その時はまだインスタントコーヒーだけでした。しかし、はまってから数か月後インスタントコーヒーと間違って抽出タイプの粉を買ってしまいました。それを飲むために安いコーヒーメーカーを買いました。そしてその粉が多かったため同じ味に飽きてしまいました。そこでコーヒーの豆を売る店を調べ（コーヒー豆を売る店が意外と多くあることに驚きました。）、豆を買いました。この時にミルも買いました。フレンチプレスという器具でも入れて飲んでみたりしました。そうして豆を挽いて飲んでいくうちに自分で入れたくなり大学4年のはじめ頃ついにヤカンとドリッパーを買い、自分で入れて飲むようになりました。もうこのころすでにインスタントコーヒーはあまり飲まなくなり、コーヒーメーカーは使わなくなりました。
コーヒーは好きですが特にこれが好きという豆は今のところありません。というかいろいろな豆で飲むことが一期一会的な感じで楽しんでいます。好みのフレーバーはコクと苦みだった。最近ではゲイシャを飲んだため、酸味もいいなと思えるようになってきた。なのでフレーバーは何でも好き。好きなドリッパーはハリオの「円錐型ドリッパーV60」です。というか円錐型以外は入れにくくて苦手です。
2018/10ごろからネルドリップに興味を持ちそれからは一日一回はネルで入れるようにしています。そしてネルで入れた香りの強いコーヒーを水筒に入れて人のいるところでコーヒーを飲んで香りテロをするのがひそかな楽しみになっています。（もちろんコーヒーの香りが苦手な人がいるとわかった時はそこではやらないようにしています。）

## デジ絵
就活が終わったので何かしようと思い、苦手を克服するために絵を描き始めたのがきっかけ。pixivで参考にしたい絵師を探す中ではまり込んでいった。そんな中、朝倉色さんの手描き水彩でスライム娘という作品を見てスライム娘にはまる。そこで絵は苦手なので何度もやり直すことができるデジタル環境でできるようにと板タブを買う。デジ絵を描くのは楽しいという経験も得ることができたが、絵を描くとはこんなにも大変なものかとトレースや模写などを通して知る経験もできた。アウトプットの重要性は知っているので発表してもよさそうな作品が描けたら発表していく。まだなにも見ずに人物画を描けるほど絵がうまいわけではないのであまり作品を発表できないので、お手柔らかにしていただきたい。最近ではデジ絵を描いたり公表するのは創作意欲がわいたときだけなので一年に数回あるかないか。でもおかげでデジ絵を始める前と比べて落書きは増えた。
